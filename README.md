# 写真からカレンダー作成アプリ

このアプリは、写真に写っている文字を OCR で読取り、内容を編集して Google Calendar に追加できるブラウザ向けツールです。

## 使い方

1. 画像をアップロードする
2. OCR ボタンで本文を読み取る
3. タイトル・日付・時間などを必要に応じて修正する
4. Google Calendar で作成ボタンを押して登録画面を開く

## 注意

- OCR はブラウザ上で実行され、画像はローカルに保存されません。
- Google Calendar の作成画面は、Google にログイン済みならそのまま登録できます。

## 実行方法

- ルートで `python -m http.server` を実行
- ブラウザで `http://localhost:8000` を開く
